# Demo Spine — Torq

## Living question we answer
> Do fans want the market’s heartbeat as a second screen?

## Void we close
“The score doesn’t feel like the match.”

## User in the first 10 seconds
Fan, TV on, phone in hand, chat exploding, doesn’t know which signal to trust.

## The aha click (product moment)
The win-probability wave lurches on a goal / market shift — felt before commentary catches up.

## Field quote on screen (optional but strong)
> “Which game should I actually be watching right now?”

## Click path (≤90s)
1. fervor.up.railway.app  
2. France vs England (live) or Replay backup  
3. Point at the wave  
4. GoalBlast / moment + Beat the market  
5. Proof badge  
6. Hover Dynamic Island  

## What we explicitly do NOT demo
- Wallet seed / crypto onboarding  
- OpenAPI / bootstrap  
- FossaPay / other products  
- Feature laundry list  

## Pitch order
1. Hook — 15s (two stories: pitch + market)  
2. Void — 30s (score ≠ feel)  
3. Workaround — 15s (five apps)  
4. Product moment — 60–90s (wave + goal + island)  
5. Proof — 20s (TxOracle + TxLINE)  
6. Ask — 10s (Consumer track / live URL)  

## Backup if live demo fails
Any FT match at 60× (France–Morocco recorded).

## Claims we will NOT make (unvalidated)
- “Fans prefer this to SofaScore” (unproven)  
- “Beats competitor X” (don’t name competitors in pitch)  
- Real-money edge or betting advice
